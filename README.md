---------------------------------------------------------------------------
------------------------------GETTING STARTED------------------------------

1. Clone the Repository
   
-> git clone <url>
-> cd sightsaver


2. Install Dependencies

-> npm install

3. Run the App

-> npm run dev

---------------------------------------------------------------------------
---------------------------------TECH STACK--------------------------------

Category - Technology

Frontend	- React (Vite)

Routing	- React Router DOM

Icons	- React Icons

UI Styling - CSS (Custom)

Calendar	- react-calendar
